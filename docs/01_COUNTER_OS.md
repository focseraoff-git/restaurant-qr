The **Counter OS** is the command center for Front-of-House operations. It allows staff to create orders, manage tables, and process payments efficiently.

## 1. Dashboard Overview
Upon entering Counter OS, you see the **Main Dashboard**.
*   **Navigation Tabs**: `New Order`, `Live Orders`, `Order History`.
*   **Status Bar**: Shows system time and connection status.

## 2. Creating a New Order (`New Order` Tab)
This panel is optimized for speed.
1.  **Select Order Type**:
    *   **Dine-In**: Requires selecting a Table Number.
    *   **Takeaway**: Requires entering a Customer Name (and optional Phone).
2.  **Add Items**:
    *   Browse the **Menu Grid** on the left.
    *   Use the **Search Bar** to find items quickly.
    *   Filter by **Categories** (Starters, Main Course, etc.) using the dropdown.
    *   Click an item to add it to the cart.
3.  **Manage Cart**:
    *   Adjust quantities using `+` / `-` buttons.
    *   View Subtotal and Tax breakdown.
4.  **Confirm Order**:
    *   Click the large **âš¡ CONFIRM ORDER** button.
    *   **Result**: The order is sent to the Kitchen instantly, and a Bill is automatically printed (popup).
